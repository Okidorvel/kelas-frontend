# CATATAN PERTEMUAN 1
Berlangsung pada 23 Maret 2017, 14.00-15.00 WIB @ [QODR HQ](https://goo.gl/maps/MBNM8tfTmd82)

## PENGENALAN
- Kurikulum pembelajaran.
- Fokus pembelajaran pada frontend design bukan frontend javascript.
- Pembelajaran dimulai dari dasar hingga bisa membuat template sendiri.
- Akan ada tugas ditiap pertemuan dan dikumpulkan melalui `github`.
- Juga akan mempelajari javascript dasar dan php dasar.
- Juga akan ada pembelajaran design (optional)
- Gambaran karir frontend developer @ [TONJOO](https://tonjoo.com/career/)
- Ada grup `Trello` untuk frontend developer seluruh cabang QODR (di-invite).

## TARGET
- Bulan pertama : membuat template dengan css native.
- Bulan kedua : membuat template dengan bootstrap css.
- Bulan ketiga : menggunakan foundation css, wordpress dan frontend tools.

## TIPS
- Diperlukan kemampuan mengetik untuk tingkat lanjut adapun pemula tidak begitu masalah.
- Penggunaan `snippets` untuk mempercepat coding tetapi tidak direkomendasikan untuk yang baru belajar.
- Website belajar mengetik cepat:
  - [TypingClub](https://typingclub.com)
  - [10FastFingers](https://10fastfingers.com)
  - [Typing](https://www.typing.com/)
- Follow frontend developer yang sudah  berpengalaman di Facebook.
